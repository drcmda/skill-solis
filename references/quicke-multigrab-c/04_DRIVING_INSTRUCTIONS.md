# 04 — Driving Instructions

> **Source:** Quicke Multigrab C / CS Grab Bucket — Instruction Manual, Ålö AB (Quicke), Umeå Sweden — document 60081299 E, 2025-01 (English section of the multilingual manual) — Chapter 04, pp.15–16.

This chapter defines the designated use of the Multigrab grab bucket, the pre-operation checks to perform before each use, the operating technique for filling, gripping and moving material, and how to park the implement safely. Because the Multigrab is an attachment for a front loader / base machine, it repeatedly refers the operator back to the loader's / base machine's user manual for connection, disconnection and actuation procedures.

## 4.1 Application areas

### 4.1.1 Designated use

The implement consists of a **bucket with grab equipment**. Its main application area is the handling of:

| Designated material |
| --- |
| Garden waste |
| Light bucket work |
| Loose silage |
| Light bales |

> **NOTE:** Carefully read through and familiarise yourself with the loader's / base machine's instruction manual regarding how to operate the implement (empty / break open, open / close, etc.) prior to use.

### 4.1.2 Before operating the implement

For connecting and disconnecting the implement, see the loader's / base machine's user manual.

Check with caution that:

- The hydraulic hoses are correctly connected and the implement moves as expected.
- There is no leakage of hydraulic oil.
- There is no risk of collision between loader arm and implement (see **Fig.17**).
- The implement is correctly fitted and that the pins are in the **locked position (1)**. Press the tip of the implement against the ground to check that the implement is **firmly fixed (2)** (see **Fig.18**).

**Hose-connection tip:** To make it easier to connect the hoses, the adapter and / or the connecting hose is connected to the cylinder's two sides as marked by colour:

| Cylinder side | Marking colour |
| --- | --- |
| (+) side | Red |
| (−) side | Blue |

A bold **downward arrow** points at the pivot / linkage pinch-point where the loader arm meets the implement — i.e. the zone to watch for collision. Caption: *Check carefully that there is no risk of collision between loader / base machine and implement.*

**Fig.18 (described):** The mounted implement (side view, set on the ground) shown with two callouts:

> ⚠️ **WARNING:** Do NOT stand, walk or work under a lifted loader arm. Make sure that you keep all people, especially children, and animals away from the working area.

### 4.1.3 Using the implement

Carefully read through and be familiar with the loader's / base machine's user manual regarding how to operate the implement (empty / break open, open / close, etc.) prior to use.

### 4.1.4 Operating instructions

- Use the **upper section** (gripper arm) to hold material in the bucket.
- **Lower the loader arm as far as possible when moving.** Keep in mind that the higher you lift the loader / base machine, the higher the centre of gravity, resulting in an **increased risk of the base machine overturning**.
- When closing the upper section, objects that tend to push the tines apart must be gripped such that this is prevented, in order to avoid damage to the implement — e.g. by using a **clamping strip**.

**Fig.20 (described):** Front (head-on) elevation of the empty grab bucket, showing the full row of vertical upper-section gripper tines, the central pivot / linkage, and the two end attachment lugs. Illustrates the layout of the upper section used to hold material. Caption: *Operating instructions.*

**Fig.21 (described):** Two front-on views of a round object (a log) held by the closed grab tines, contrasting incorrect vs. correct handling:
- **Marked with an ✗ (incorrect):** the log sits on a flat bucket edge and **forces the tines apart** — small burst / motion lines either side of the log show the tines being spread.
- **Marked with a ✓ (correct):** the same log rests on the **serrated / scalloped lower bucket edge** with the tines held in their proper position, so the object is not allowed to push them apart.

Caption: *Objects must NOT be allowed to push the upper section tines apart.*

### 4.1.5 Parking the implement

> **IMPORTANT:** Position the implement on a level and solid surface with the upper section (gripper arm) in the **lowered position**.

> **IMPORTANT:** Do not disconnect the implement with the hydraulic cylinders in the **end position** — difficulties may arise due to overpressure the next time the quick couplings are to be connected.

See also the connection and disconnection instructions in the loader's / base machine's user manual.

**Fig.22 (described):** Side view of the Multigrab set down on the ground in the parked configuration, resting on a level solid surface with the gripper arm (upper section) lowered onto the bucket. Caption: *Parking the implement.*
