# Quicke Multigrab C / CS — Grab Bucket (Front-Loader Attachment)

> **Source:** Quicke Multigrab C / CS Grab Bucket — Instruction Manual, **Ålö AB (Quicke)**, Umeå, Sweden — document **60081299 E**, 2025-01 (English section of the 14-language manual). 7 chapters.

This is the reference hub for the **Quicke Multigrab C / CS** — a *grab bucket* (Greifschaufel): a bucket fitted with a hydraulically-operated upper "grab" arm. It is an **attachment** that mounts on the front loader of a base machine (such as the Solis 26 tractor). It is **not** a standalone machine — its weight and load limits are governed by the base machine.

> ⚠️ **Always read the base machine's (loader/tractor) manual as well.** IMPLEMENT WEIGHT + LOAD WEIGHT must never exceed the base machine's permitted limit. The driver is responsible for safe operation.

---

## What it is

- A **bucket with grab equipment** for **compact loaders** with a lifting force of **300–1400 kg**.
- Two series / three models: **Multigrab CS** (model **CS 120**) and **Multigrab C** (models **CM 140**, **CL 170**).
- Main uses: garden waste, light bucket work, loose silage, light bales.
- Optional accessories: **bolt-on gables** (increase bucket volume) and a **bolt-on clamp edge** (hold material without puncturing it, e.g. bales).
- Hydraulics: dual-action cylinder(s); **max working pressure 175 bar / 17.5 MPa** (use a pressure-restriction kit above 175 bar). Hose colour code: **(+) side = red, (−) side = blue**.

## Quick safety essentials

- Signal words: **DANGER** (death/serious injury) › **WARNING** (could cause serious injury/death) › **CAUTION** (could cause injury) › **IMPORTANT** (property/process) › **NOTE** (info).
- Never use as a **working platform** or to **lift/transport people**. Keep people, children and animals clear of a raised loader arm; never stand under it or between the base machine and the loader cross tubes.
- **Pressurised oil** can inject through skin (invisible leaks): use safety glasses + leather gloves, detect leaks with cardboard/wood — **never fingers/hands** — and get a doctor within hours if injected.
- **Lubricate every 10 operating hours** with universal grease at the articulation-pin nipples (both sides); lower the implement to the ground before lubrication.
