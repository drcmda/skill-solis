## 1.1 General information

> ⚠️ **WARNING:** Careless or incorrect use may result in serious injury or death for the driver or others. Observe the safety instructions.

This instruction manual describes the operation and maintenance of the implement. The information in the manual was correct when it went to press. Visit your dealer if there is anything in the manual you do not understand.

A user manual is supplied with each implement to show the operator how to handle and maintain it. Read and use the information so you can operate the implement safely, with only short stoppages. The implement is designed to be simple to service, allowing maintenance with regular hand tools.

**Before using the implement:**
- Read and study the text in the user manual thoroughly before you start to use the implement.
- If you are not an experienced driver, study the user manual and ask an experienced driver to explain things to you. Your dealer can teach you about operation and suitable work methods.
- Keep this user manual readily available — preferably inside the base machine.
- Get a new user manual if the old one is damaged or lost.
- Read the user manual for the loader / base machine too; it contains important information for your safety.

> **CAUTION:** Read through the entire instruction manual before you start to use the product.

> **NOTE:** Ålö AB reserves the right to introduce changes to design and specification, or improvements, at any time, without prior notice or commitment.

## 1.2 Type plate (machine sign)

Fill in the information from the implement's type plate and always state it when contacting your dealer. The sign specifies: **Product, Type, Reference number, Serial number** and **Weight**.

> **IMPORTANT:** Always provide **Product (A)**, **Type (B)** and **Serial number (C)** when contacting your dealer for service issues or spare-parts needs.

### Read-the-manual symbol (Fig.1)

**Fig.1** — Open-book pictogram. Meaning: *Read through the entire user manual before you start to use the product.*

> **Note (corrected):** `A4`, `A5` and `A6` are **separate barcoded fields in their own cells** — they are **not** alternative callout names for KU / AG / CN. (The North-America plate has **no Kit No.** field; the Kit No. field appears only on the standard Fig.3 plate.)

**Manufacturer / marks (on all signs):** Ålö AB, Umeå Sweden, "Made in …", with the **Quicke "Q" logo**, an **ET Certificate** quality mark and the **CE** and **UKCA** conformity marks — and reseller / dealer information recorded alongside.
