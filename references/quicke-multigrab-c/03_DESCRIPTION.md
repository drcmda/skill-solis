## 3. Description

Multigrab C and Multigrab CS are intended for compact loaders with a lifting force between **300–1400 kg**.

The implement consists of a **bucket fitted with grab equipment**.

| Property | Value |
| --- | --- |
| Implement type | Grab bucket (bucket with grab equipment) |
| Intended base machine | Compact loaders |
| Lifting force range | 300–1400 kg |
| Variants covered | Multigrab C, Multigrab CS |

> **NOTE:** To check which lifting force is appropriate for this manual's intended implement, see **6.1 Table of recommendations** (Chapter 6).

## 3.1 Accessories — Bolt-on gables

Bolt-on gables, which **increase the bucket's volume**, are available for the implement.

The instructions supplied with the kit describe how to assemble these bolt-on gables.

**Fig.15 — Bolt-on gables are available for the implement.** The figure shows the grab bucket in side/three-quarter view with raised bolt-on side panels (gables) added to the left and right ends of the bucket. The gables extend the side walls upward and outward beyond the standard bucket profile, enlarging the load-carrying volume of the bucket. The upper grab tines/jaw assembly remains in place above the enlarged bucket body.

> **NOTE:** Assemble the bolt-on gables according to the instructions supplied with the gable kit.

## 3.2 Accessory — Bolt-on clamp edge

The upper section of the attachment can be equipped with a **bolt-on clamp edge** to extend the implement's area of application. The strip can be used to avoid poking holes in the material, e.g. when handling bales.

The instructions supplied with the kit describe how to assemble the bolt-on clamp edge.

**Fig.16 — The implement can be fitted with a clamp edge.** The figure shows the grab bucket with the upper grab section (top jaw) fitted with a bolt-on clamp edge: a continuous flat strip/bar running along the gripping edge of the upper jaw. Instead of pointed tines biting into the load, the clamp edge presents a smooth, full-width clamping surface that holds the material (such as bales) firmly without puncturing it, broadening the range of materials the implement can handle.

> **NOTE:** Assemble the bolt-on clamp edge according to the instructions supplied with the clamp-edge kit.
