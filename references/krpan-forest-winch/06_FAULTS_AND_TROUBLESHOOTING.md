# 06 — Improper Use, Troubleshooting & Quick Fixes

> **Source:** KRPAN® 4 EP / 5, 6, 8 ERP Single-Drum Forest Winch — Operating Manual (Návod k obsluze / "Návod na použití a bezpečnou práci"), KRPAN d.o.o. (Slovenia), revision 03.02.2025 — translated from the Czech edition — pp.34–36.

This reference covers what goes wrong when the single-drum mechanical forest winch (jednobubnový lesní mechanický naviják) is misused, the safe procedure for any intervention on the machine, and a problem → cause → remedy table for the most common faults. The winch mounts on the tractor's three-point linkage and is driven by the PTO / cardan shaft (kardanová hřídel) at 540 rpm.

## 9. Consequences of Improper Use of the Forest Winch

If the winch is used improperly, the consequences may be:

- a burnt-out clutch (spálená spojka),
- a burnt-out brake (spálená brzda),
- a damaged brake mechanism (poškozený mechanizmus brzdy),
- a worn-through / stripped link chain (utřený článkový řetěz),
- a broken pulley or pulley bearing (zlomená kladka nebo ložisko kladky),
- damage to the cover/guard of the PTO / cardan shaft (poškození krytu kardanové hřídele),
- damage to the gear wheel (poškození ozubeného kola),
- damage to the housing/cover or destruction of the frame (poškození krytu, resp. destrukce rámu),
- a snapped / torn-off rope (utržené lano),
- destruction of the drum axle (destrukce osy bubnu), …

## 10. Troubleshooting / Servicing the Machine — Safe-Intervention Procedure

For any intervention on the machine (replacing the chain, tensioning/replacing the belt, changing the oil, routine cleaning of the machine, etc.) the following steps must be observed:

- **SWITCH THE MACHINE OFF! (STROJ VYPNĚTE!)**
- Disconnect the cardan shaft (kardan) from the tractor.
- Wait until the machine has cooled down, so that you do not burn yourself on hot parts when reaching into the machine.
- Once you have made sure the machine is sufficiently cooled, you may begin the work.
- Once you finish the work, reassemble the machine back to its original state so that working with it is safe again.
- Then you may switch the machine on again and continue working.
- In case of any difficulties, contact our service centre (servisní středisko).

> ⚠️ **WARNING:** For every intervention on the machine, the machine must be switched off and the drive (cardan shaft / kardan) must be disconnected.

> 🛑 **DANGER:** **Any intervention on the machine while it is running is forbidden!!! (Jakkoli zasahovat do stroje za provozu je zakázáno!!!)**

> ⚠️ **WARNING:** If welding (sváření) on the forest winch is a possibility, the control console (ovládací konzola) must be disconnected.

## 11. What to Do When… (Problem → Probable Cause → Remedy)

| PROBLEM | PROBABLE CAUSE | REMEDY |
|---|---|---|
| **Winch does not pull / drag hard enough** (Naviják nevleče dostatečně) | Too much pulling rope on the drum (příliš mnoho vlečného lana na bubnu) | Check the maximum permitted rope length on the drum. |
| | Incorrect clutch (spojka) adjustment | Adjust the clutch. |
| | Greasy / contaminated clutch plate-lamella (zamazaná lamela spojky — from incorrect lubrication of the drive chain) | Replace the clutch lamellae / friction plates (výměna spojkových lamel). ** |
| | Worn clutch lamella (opotřebená lamela spojky) | Replace the clutch lamellae. ** |
| | Burnt-out clutch disc / plate (spálený spojkový talíř) | Replace the clutch lamellae. ** |
| | Damaged clutch disc / plate (poškozený spojkový talíř) | Replace the clutch lamellae. ** |
| | Damaged drive component of the winch (poškozený pohonný díl navijáku) | The damaged part of the winch must be replaced. |
| **Brake does not hold** (Brzda nedrží) | Incorrect brake (brzda) adjustment | Adjust the braking force / output (nastavte brzdný výkon). |
| | Greasy / contaminated brake lining (zamazané obložení brzdy) | Replace the brake belt / band (výměna brzdového řemene). ** |
| | Damaged brake mechanism (poškozený mechanizmus brzdy) | Replace the necessary damaged parts. ** |
| | Worn brake (opotřebená brzda) | Replace the brake belt / band. ** |
| **Pulling rope is hard to pull out / pay out** (Vlečné lano se těžce vyvléká) | Incorrect setting of the rope pay-out force (nesprávné nastavení síly vyvlékání lana) | Adjust the rope pay-out force. |
| | Damaged pulling rope (poškozené vlečné lano) | Replace the pulling rope. ** |
| | Damaged brake (poškozená brzda) | Replace the brake belt / band. ** |
| | Dirt / contamination inside the winch housing (nečistota v plášti navijáku) | Unwind the pulling rope and clean the winch. |
| **Winch pulls even though the clutch is disengaged** (Naviják vleče i přes vypnutou spojku) | Incorrect clutch (spojka) adjustment | Adjust the clutch. |
| | Damaged drum (poškozený buben) | Replace the damaged drum. |
| | Damaged clutch (poškozená spojka) | Replace the damaged clutch. ** |

> **NOTE:** ** = consumable material (spotřební materiál).

> **IMPORTANT:** More demanding repair work on the winch must be carried out by a specialist (the service department).

> ⚠️ **WARNING:** The winch is tested for performance and safety. For trouble-free and safe operation, only original spare parts must be used in the event of a fault.

> ⚠️ **WARNING:** The buyer loses all warranty rights if non-original spare parts are used, or if a repair is carried out improperly or by an unauthorised person.
